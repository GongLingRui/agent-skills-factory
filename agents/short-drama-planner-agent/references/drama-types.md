# 常见竖频短剧类型与受众分析

## 一、男频主流类型

### 1. 战神/赘婿逆袭
- **核心爽感**：身份落差带来的碾压式反击
- **典型结构**：废物身份受辱 → 真实身份揭露（战神/神医/首富）→ 仇人跪舔
- **目标受众**：25-45 岁男性，三四线城市为主
- **情绪关键词**：解气、扬眉吐气、权力幻想
- **投流素材方向**：「被丈母娘赶出门」→「直升机降落喊主帅」
- **风险提示**：同质化严重，需在人设或反转方式上做微创新

### 2. 重生/穿越复仇
- **核心爽感**：预知未来的降维打击
- **典型结构**：前世惨死 → 重生回到关键节点 → 步步为营反杀仇敌
- **目标受众**：20-40 岁男女均衡
- **情绪关键词**：痛快、智商碾压、弥补遗憾
- **投流素材方向**：「前世被闺蜜推下天台」→「重生回到婚礼前一天」

### 3. 都市神豪/系统流
- **核心爽感**：金钱/能力的无限膨胀
- **典型结构**：获得系统/继承遗产 → 花钱升级/完成任务 → 征服各领域
- **目标受众**：18-35 岁男性
- **情绪关键词**：暴富、奢华、为所欲为
- **投流素材方向**：「被女友嫌弃穷」→「黑卡限额 10 亿」

### 4. 悬疑探案/惊悚
- **核心爽感**：抽丝剥茧的智力快感
- **典型结构**：诡异案件发生 → 主角发现异常 → 层层反转揭露真相
- **目标受众**：25-40 岁，男女均衡，一二线城市占比高
- **情绪关键词**：烧脑、细思极恐、反转
- **投流素材方向**：「直播算命，算一个死一个」

## 二、女频主流类型

### 1. 甜宠/霸道总裁
- **核心爽感**：被极致宠爱的情感满足
- **典型结构**：平凡女主遇到高冷男主 → 误会与试探 → 男主独宠女主
- **目标受众**：18-35 岁女性，全城市线覆盖
- **情绪关键词**：甜、上头、姨母笑
- **投流素材方向**：「误闯总裁办公室」→「全公司喊她老板娘」
- **变体方向**：先婚后爱、契约婚姻、替嫁、闪婚

### 2. 大女主复仇/宅斗
- **核心爽感**：女性觉醒后的反击与独立
- **典型结构**：被陷害/被背叛 → 黑化/觉醒 → 以智谋或实力让仇敌付出代价
- **目标受众**：25-45 岁女性
- **情绪关键词**：飒、解气、女性力量
- **投流素材方向**：「被丈夫和小三逼到绝路」→「华丽归来收购前夫公司」

### 3. 穿越/古言
- **核心爽感**：现代思维在古代的降维打击
- **典型结构**：现代女主穿越 → 用现代知识/价值观破局 → 收获爱情与权力
- **目标受众**：18-35 岁女性
- **情绪关键词**：新鲜、共鸣、浪漫
- **投流素材方向**：「穿越成废材庶女」→「用化学知识制冰震惊全场」

### 4. 家庭伦理/婆媳
- **核心爽感**：道德高点上的反击与和解
- **典型结构**：女主隐忍付出 → 被婆家/丈夫误解欺压 → 爆发或逆袭
- **目标受众**：35-55 岁女性，下沉市场为主
- **情绪关键词**：委屈、解气、亲情
- **投流素材方向**：「婆婆逼她离婚」→「她竟是失散多年的豪门千金」

## 三、类型融合与创新方向

| 融合方向 | 示例 | 优势 |
|---------|------|------|
| 甜宠 + 悬疑 | 霸道总裁有双重身份，女主逐步揭开 | 兼顾女性向情感与智力爽感 |
| 战神 + 家庭 | 退役战神回归家庭，保护妻女 | 扩大受众年龄层 |
| 重生 + 职场 | 重生回到职场关键节点，商战复仇 | 都市白领共鸣强 |
| 系统 + 短剧 | 绑定"短剧系统"，完成剧情任务获奖励 | 元设定有趣，易出梗 |

## 四、平台特性差异

| 平台 | 用户画像 | 偏好类型 | 内容调性 | 审核重点 |
|------|---------|---------|---------|---------|
| 抖音 | 年轻、女性偏多 | 甜宠、逆袭、悬疑 | 节奏更快、视觉更炸 | 价值观导向、软色情 |
| 快手 | 下沉、男性偏多 | 战神、神豪、家庭伦理 | 更接地气、更直接 | 暴力、拜金 |
| 视频号 | 中老年、女性 | 家庭伦理、甜宠 | 温情、正能量 | 政治、迷信 |
| 红果/番茄 | 小说用户转化 | 与原著类型一致 | 还原度高 | 版权、改编尺度 |
