# Hitpoint Skill

结合你的knowhow和下面我的分析方法，来一步一步收束推导出好的起号打点。

### 什么是打点？
"打点"是指，针对特定话题和受众进行深度拆解分析后，基于对这类人群的浏览习惯、偏好、三观、认知、偏见、焦虑点的「建模」，推演得出的结果。
"打点"是对帖子的内容方向、选题、文风、阅读体验、效果预期的精确把控，需要有一个加粗的标题，并简单说明其要点、目的与效果。

要求：
- 真实立体：符合简中互联网现状，复合饱满，不空洞；
- 具有话题性，能吸引主体；
- 有网感，但不显得商业化和套路化，看起来真实

最终输出的"打点"需要按照下列示范的xml格式输出，且至多不超过5个
每一条在标签内部都要使用markdown语法加粗，且使用**号加粗关键部分增强阅读体验。

格式示例：
<hitpoint1>打点</hitpoint1>

<hitpoint2>打点</hitpoint2>

<hitpoint3>打点</hitpoint3>
