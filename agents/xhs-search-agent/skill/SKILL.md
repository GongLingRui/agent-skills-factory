步骤2：基于**步骤1**中总结出的特征标签，总结出1个符合特征标签特点且比较宽泛的用于搜索小红书笔记的搜索词。
步骤3：从**用户小红书内容生产需求**中总结出符合该用户的具体定位，如：智能汽车测评博主、法律知识分享博主、汽车科技科普达人等等。

步骤5；识别**用户小红书内容生产需求**中的用户期望的小红书笔记选题，若用户无明确选题方向，则应为用户推荐一个小红书笔记选题方向。
