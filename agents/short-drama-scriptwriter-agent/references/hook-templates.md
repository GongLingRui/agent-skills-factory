# 黄金三秒钩子模板库

> 竖频短剧前 3 秒决定 70% 的留存率。以下钩子模板可直接套用或改编。

---

## 一、冲击型钩子（视觉/听觉强刺激）

### 模板 1：极端情绪开场
```
【画面】女主满脸是血/婚纱被撕烂/跪在暴雨中
【台词】"我这一生，最后悔的就是认识你。"
【切黑屏】+ 音效（玻璃碎裂/雷声/尖叫）
```

### 模板 2：奇观式开场
```
【画面】直升机群掠过城市/豪车连环相撞/古宅大门缓缓打开
【字幕】"三年了，他终于回来了。"
```

### 模板 3：冲突前置
```
【画面】一个响亮的耳光，主角被打得偏过头
【台词】"这一巴掌，打你不知好歹！"
【画面】主角缓缓转回头，眼神骤冷（特写）
```

## 二、悬念型钩子（引发好奇）

### 模板 4：倒计时/预言
```
【画面】手机屏幕显示倒计时 00:03:00
【台词】"三分钟后，这栋楼会塌。"
【画面】周围人嘲笑，主角冷静看手机
```

### 模板 5：身份错位
```
【画面】保洁阿姨正在擦地
【台词】"妈，您别擦了，这公司都是您的。"
【画面】秘书慌张冲进来："董事长，股东们等您开会！"
```

### 模板 6：死亡开场
```
【画面】棺材/墓碑/火葬场
【台词】"她死的那天，全城下了三天三夜的雨。"
【画面】一只手从土里伸出来 / 某人猛地睁开眼
```

## 三、反转型钩子（预期违背）

### 模板 7：身份反转
```
【画面】婚礼现场，新郎掀桌："这婚我不结了！"
【画面】新娘淡定鼓掌："巧了，我也不打算嫁。"
【画面】新娘摘下假发，露出真实身份（特工/总裁/仇人）
```

### 模板 8：强弱颠倒
```
【画面】大佬带着百人围住一个小伙
【台词】"今天就是你的死期。"
【画面】小伙打个电话，直升机降临，下来的全是顶级特种兵
【台词】"你确定？"
```

### 模板 9：认知颠覆
```
【画面】学霸被全班嘲笑考了 0 分
【画面】校长亲自冲进教室："你的试卷是国家级机密，我们无权批改！"
```

## 四、情感型钩子（快速共情）

### 模板 10：亲情暴击
```
【画面】小女孩在雨中卖花，被豪车溅了一身泥
【画面】车窗摇下，里面坐着她的亲生父亲，却没认出她
【台词】"爸爸，买朵花吧……"
```

### 模板 11：背叛瞬间
```
【画面】女主端着蛋糕准备给男友惊喜
【画面】推开门，男友和闺蜜在一起
【画面】蛋糕掉在地上，女主的表情从笑到凝固（慢镜头）
```

### 模板 12：重逢错位
```
【画面】男主在人群中一眼认出女主的背影
【画面】追上去拍她肩膀，她转身，是一张陌生的脸
【画外音】"三年了，我还是会在每个人身上找你的影子。"
```

## 五、 Cliffhanger 结尾模板（逼观众点下一集）

| 类型 | 模板示例 |
|------|---------|
| 身份即将揭露 | 反派举起枪："你到底是谁？" 主角摘下面具，反派瞳孔骤缩——黑屏 |
| 关系质变 | 女主踮脚要吻男主，男主突然推开她："我不能。" —— 画面定格在女主错愕的脸 |
| 危机降临 | 主角刚解决完所有敌人，手机收到一条短信："你以为结束了？抬头。" —— 他抬头，更大危机 |
| 真相碎片 | 主角发现一张照片，上面是死去的母亲抱着一个婴儿——那个婴儿不是他 —— 黑屏 |
| 实力展现 | 众人逼迫主角出手，主角一直拒绝。最后某人动了他在乎的人，他缓缓卷起袖子，露出龙纹 —— 集终 |
| 选择困境 | 救 A 还是救 B？倒计时 10 秒。主角看向 A，又看向 B —— 黑屏 |

## 六、钩子组合策略

**高留存集的结构公式**：

```
第 N 集结尾 cliffhanger（悬念钩子）
  ↓
第 N+1 集开头 3 秒：冲击型钩子（承接悬念，给一个强刺激）
  ↓
第 N+1 集 15 秒：悬念型钩子（埋新钩子）
  ↓
第 N+1 集 45 秒：情感型钩子（拉共情）
  ↓
第 N+1 集结尾：更强的 cliffhanger
```

**投流素材剪辑建议**：
- 取「冲击型/反转型钩子」+ 3-5 秒高潮片段
- 封面文案用悬念型钩子的台词
- BGM 在 0-1 秒突然切入高潮副歌，制造听觉冲击
